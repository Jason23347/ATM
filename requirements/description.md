# 项目描述

## 产品的前景
    该产品使我们开发的第一代产品，据我们调查，我们开发的ATM自动存取款系统用户界面十分友好，后台功能强大，有较强的市场竞争力，符合用户对快速存取款的需求。

    这个项目的开发是为银行提供一套高效稳定可靠的终端服务平台，为储户登录、存款、取款，查询等操作提供便利。
